Mempool Info

The mempool holds unconfirmed transactions waiting to be included in the next block.

In testnet mode, blocks are produced on demand via the produce_block RPC call or the button above.

Testnet Faucet

Get free testnet tokens (TNKL or sUSDC) to start experimenting with shielded transfers. Tokens have zero real-world value and are for testing only.

1
Create a wallet (if you haven't already)
./tonkl wallet
The setup wizard runs automatically on first launch. It will generate your seed phrase and first key.
2
Request testnet TNKL tokens
./tonkl wallet faucet
Drips 100 TNKL to your default key. One request per hour per address.
3
Send a shielded transfer
./tonkl wallet send 50
Sends 50 TNKL as a zero-knowledge shielded transaction. Balances, amounts, and identities stay hidden on-chain.
Other Assets

You can also request sUSDC (Shielded USDC) for multi-asset testing:

./tonkl wallet faucet --asset-id 4
